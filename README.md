Manipular arrays em C, é algo essencial, para que seja possível, escrever programas robustos e muito interessantes, é válido destacar, que
declarar um array em C, é o mesmo que declarar um ponteiro, que aponta para a posição inicial desse array, e que consegue apontar
para os elementos seguintes, considerando a existência de blocos de memória contíguos, o que nos remete ao princípio da localidade espacial
(ou seja ao acessar um endereço de memória, de um bloco específico, o ponteiro aponta para o próximo).
